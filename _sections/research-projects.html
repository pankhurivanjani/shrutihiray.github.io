Lightning Network Simulator (Dual Degree Thesis)

  • Studied scalability for Bitcoin blockchain with Lightning Network.
  • Implemented a simulator for multihop payment routing in Lightning Network in Haskell & Python.
  • Analyzed Flare routing protocol - achieves 98% reachability to any node in the 2000 node network with only 4 beacons.
