Some of the projects I have worked on:

Multihop Payment Routing in Lightning Network (Dual Degree Thesis)

  • Studied Lightning Network as a layer built on top of Bitcoin blockchain.
  • Implemented a simulator for multihop payment routing in Lightning Network in Haskell.
  • Analyzed Flare routing protocol - achieves reachability to any node with high probability with minimum routing state

Multisource and Multipath File Transfer

  • Utilisation of multiple network interface in devices to download file from multiple sources.
  • Implemented browser plugin for simultaneous download through Cellular and WiFi.
  • Parts of file downloaded from cache in WiFi access points reduces network congestion considerably.
  • Increases throughput of file download.

Wavelets

Wireless Communication

Speech Watermarking

  • Embedding of hidden data in host signal and extraction of original signal at target.
  • Implemented using Autoregressive model and Psychoacoustic model.
  • Validation using robustness, perceptual transaparency and capacity.

Speech Recognition of digits

  • Classified speech signals of digits using labelled data.
  • Implemented using Cepstrum, K-means clustering and Dynamic time warping.

Pattern Recognition of remotely sensed images

  • Pattern recognition of multispectral satellite images using neural networks.

Commercial success prediction of a song

  • Classification of success of song by feature extraction, PCA and machine learning algorithms of GBM & Random Forests.
  • Used features of metadata (release date, artist, etc.) and music data (loudness,pitch, timbre, etc.)

EEG Signal Acquisition

  • Designed analog front-end for single channel, low cost, Electroencephalographic(EEG) signal acquisition.
  • Used techniques of amplification, filtering noise and analog to digital conversion.

Bicycle short animation film

  • Used computer graphics techniques of 3D transformations, backface culling, texture mapping, gouraud shading etc.
  • Implemented hierarchical modelling, modelling-viewing pipeline in OpenGL.

Microprocessor Design

  • Designed a 16 bit - 8 register RISC computer system of 14 instructions.
  • 6 staged pipeline processor which includes instruction fetch, instruction decode, register read, execute, memory access, and write back.
  • Implemented hazard mitigation technique using data-forwarding mechanism

Zombie Apocalypse Simulator

  • The application simulated the behavior of zombies and humans based on parameters such as death age, strength, food, zombie chase radius, birth rate, etc.
  • Implemented Ant Colony Optimization algorithm and SDL 2.0 library.
